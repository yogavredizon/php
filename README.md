# How to install 
1. Download this repository or using git clone to store in your computer,
2. move or copy this repository and place into C:\xampp\htdocs
2. Open Xampp, start Apache and MySql. if it's turn on, Open http://localhost/phpmyadmin/
3. Click Import Menu and chose file 'warehouse_db.sql'
4. Scroll down and klik import
5. Open your app by type http://localhost/Soal-Tugas-Materi-Kuliah-Ke-10
